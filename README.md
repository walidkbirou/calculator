# 🧮 Mini Calculator Project

A simple calculator application built with **HTML, CSS, and JavaScript**.  
This project demonstrates the use of basic front-end technologies to perform arithmetic operations and create an interactive UI.

---

## 🚀 Features
- Perform basic operations: **Addition (+), Subtraction (−), Multiplication (×), Division (÷)**
- **Clear (C)** button to reset calculations
- **Responsive design** for different screen sizes


---

## 🛠️ Technologies Used
- **HTML5** → structure of the calculator  
- **CSS3** → styling and layout  
- **JavaScript (ES6)** → logic and interactivity  

---
